# Reproduction statique - Page d'accueil

Fichiers créés:
- `index.html` : structure HTML de la page
- `css/styles.css` : styles principaux (typographie, couleurs, layout)

Comment ouvrir:
1. Ouvrez `index.html` dans votre navigateur (double-clic ou `Ctrl+O`).

Hypothèses et limitations:
- Reproduction statique simple — images et logos approximés par du texte.
- Polices servies par Google Fonts; icons très simplifiées.
- Effets et positionnements principaux respectés (couleurs, carte de signup, grille décorative).

Prochaines étapes possibles:
- Remplacer les logos par des images réelles.
- Rendre le formulaire fonctionnel avec JavaScript.
